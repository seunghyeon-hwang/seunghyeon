## Main Project

### 축구 유니폼 가격 예측 모델

평소 관심 있던 축구 유니폼 거래 시장을 주제로, 중고 거래 플랫폼 간 가격 차이와 상품 조건에 따른 가격 변동을 분석하고 예측하는 프로젝트를 진행했습니다.

웹 크롤링을 통해 거래 데이터를 수집하고, 시즌, 사이즈, 상태, 브랜드, 판매 여부 등 다양한 특성을 바탕으로 가격 예측 모델을 구축했습니다.

초기에는 단일 모델 중심으로 접근했지만, 이후 예측 성능을 비교하기 위해 여러 머신러닝 모델을 추가했습니다.

사용한 모델은 다음과 같습니다.

- Random Forest Regressor
- Gradient Boosting Regressor
- Ridge Regression
- K-Nearest Neighbors Regressor
- TensorFlow 기반 Deep Learning Model

단순히 하나의 예측값만 제공하는 방식이 아니라, 모델의 오차를 고려해 예측 가격을 범위 형태로 표시하도록 개선했습니다. 이를 통해 실제 중고 거래 가격의 변동성을 더 현실적으로 반영하고자 했습니다.

## Strengths

### 1. 문제를 직접 정의하고 구현하는 역량

주어진 과제를 수행하는 것에 그치지 않고, 일상에서 발견한 문제를 데이터 기반 프로젝트로 발전시키는 데 관심이 있습니다. 축구 유니폼 가격 예측 프로젝트 역시 개인적인 관심사에서 출발했지만, 데이터 수집, 전처리, 모델 학습, 결과 출력까지 하나의 서비스 형태로 구현하고자 했습니다.

### 2. 모델을 비교하며 개선하는 태도

처음부터 완성도 높은 모델을 만드는 것보다, 여러 모델을 실험하고 결과를 비교하며 개선하는 과정을 중요하게 생각합니다. Random Forest, Gradient Boosting, Ridge, KNN, 딥러닝 모델을 비교하며 각 모델의 특성과 한계를 이해하려고 했습니다.

### 3. 사용자 관점의 결과 표현

예측 모델의 결과가 단순한 숫자로만 제시되면 실제 활용성이 떨어질 수 있다고 생각했습니다. 따라서 예측값을 하나의 가격이 아닌 범위로 제시하여 사용자가 더 직관적으로 결과를 이해할 수 있도록 개선했습니다.
