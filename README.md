- Random Forest, Gradient Boosting, Ridge, KNN, TensorFlow 딥러닝 모델 성능 비교
- MAE를 기준으로 예측 가격 범위 제공
- Streamlit 기반 웹앱 구현 및 배포

#### Tech Stack
